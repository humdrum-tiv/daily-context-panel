# On This Day Panel

An [Obsidian](https://obsidian.md) plugin that shows a side panel with files created or modified on the same date as your active note.

On a daily note (`YYYY-MM-DD` filename), it uses that date. On any other note, it uses the note's own creation date.

## What it shows

- **Created same day** — notes created on that date
- **Last modified same day** — notes last modified on that date
- **Backlinks** — notes that link to the active note
- **Outgoing links** — notes the active note links to
- **External links** — URLs found in the active note

All sections are individually toggleable and reorderable in Settings.

## Installation

This plugin is not yet in the Obsidian community plugin directory. Install it manually:

1. Download `main.js`, `manifest.json`, and `styles.css` from this repo
2. In your vault, create the folder `.obsidian/plugins/daily-context-panel/`
3. Copy the three files into that folder
4. Open Obsidian → Settings → Community plugins → enable **On This Day Panel**
5. Click the calendar icon in the ribbon, or open it from the command palette
